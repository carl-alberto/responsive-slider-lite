== Description ==

A lightweight responsive slider utilizing bootstrap

== Installation ==

1. Upload `responsive-slider-lite.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode

== Frequently Asked Questions ==

= Why are the images not appearing=

You need to put the shortcode first in your content and enable the plugin
